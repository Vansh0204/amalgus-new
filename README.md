# AmalGus — World's First B2B2C Glass & Allied Products Marketplace 💎

> **"India's Smartest Gateway for Architectural Glass Solutions"**

AmalGus is a niche digital marketplace designed to bridge the structural gap in the glass industry. By connecting Homeowners, Architects, Builders, and Dealers directly with verified manufacturers and service partners, we bring transparency, speed, and AI-driven precision to a traditionally fragmented sector.

## 🚀 Key Features

### Core Experience ✅
- [x] **Smart Role Selection**: Tailored UX for Homeowners, Architects, Builders, and Dealers.
- [x] **Glass Catalog**: Advanced filtering by thickness, application, and processing.
- [x] **Product Detail Pages**: Technical specs, tags, and suggested hardware pairings.
- [x] **Estimate Generator**: Millimeter-to-Sqft conversion with GST and Role-based discounts.

### AI & Market Intelligence ✅
- [x] **Smart AI Matcher**: Describe your requirement in plain English and get technical glass specs.
- [x] **Daily Rates Dashboard**: Real-time market pulse marquee with trend sparklines.
- [x] **Multi-Vendor Comparison**: Compare live prices from 3+ factories per product.

### Partner Ecosystem ✅
- [x] **Vendor Directory**: Vetted manufacturing partners across major Indian cities.
- [x] **Service Partners**: Direct access to Glaziers, Installers, and Measurement teams.

## 💻 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/amalgus.git
cd amalgus
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up Environment Variables
Create a `.env.local` file in the root:
```env
NEXT_PUBLIC_OPENAI_KEY=your_key_here
```

### 4. Run Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the application.

## 📜 Architecture Overview
AmalGus leverages **Next.js 15 Route Handlers** for a lightweight backend, using local JSON persistence for the prototype phase. The frontend utilizes **React Context API** for global state management (Roles & Toasts) and **Tailwind CSS** for a futuristic, glass-morphic design language.
